# Математические формулы
Библиотека для языка программирования Python, которая умеет находить площадь и периметр треугольника.
## Площадь
### Треугольник
Для треугольника данная библиотека будет использовать всем известную формулу нахождения площади треугольника:
- Треугольник: S = (a × h) / 2
Чтобы воспользоваться данной функцией нужно прописать def area(a,h), где a - основание треугольника, h - высота треугольника, возвращаемое значение - площадь треугольника
Разберем на простом примере:
a = 10, h = 5 def area(a,h) вернет нам значением 25.0
## Периметр
### Треугольник
Для треугольника данная библиотека будет использовать всем известную формулу нахождения периметра треугольника:
- Треугольник: P = a + b + c
Чтобы воспользоваться данной функцией нужно прописать def perimeter(a,b,c), где a, b, c - стороны треугольника, возвращаемое значение - периметр треугольника
Разберем на простом примере:
a = 1, b = 1, c = 1 def perimeter(a,b,c) вернет нам значением 3
## Unit-тесты
Для проверки правильности работы функций библиотека использует встроенный модуль Python unittest.
Unit-тесты позволяют автоматически убедиться, что функции area(a,h) и perimeter(a,b,c) возвращают верные результаты при разных входных данных.
### Что проверяют тесты
Тесты запускают функции с различными параметрами и сравнивают полученный результат с ожидаемым.
Проверяются следующие случаи:
обычные значения параметров;
параметры, равные нулю;
дробные значения (float);
отрицательные аргументы.
## Запуск тестов
Команда:
python3 -m unittest triangle.py
## Ожидаемый результат
Ran 8 tests in ...
OK
